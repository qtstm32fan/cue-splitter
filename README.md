# cue_splitter

Split cue files to flac using ffmpeg.

[https://raw.githubusercontent.com/qtstm32fan/cue-splitter/master/cue_splitter.py](https://raw.githubusercontent.com/qtstm32fan/cue-splitter/master/cue_splitter.py)

```
curl -LO https://raw.githubusercontent.com/cue-splitter/cue-splitter/master/cue_splitter.py
chmod +x
cue_splitter.py -o ./out ./music.cue
```
